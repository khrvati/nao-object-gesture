\subsection{\-Detailed \-Description}
\-Class used to test trajectories for the presence of gestures. 

\-Gestures are integer lists with each element in range \mbox{[}0,7\mbox{]}. \-Each list element signifies a direction. \-A gesture is detected when a continuous segment of the tested trajectory corresponds to all of the direction elements in the list. \par
 \-Directions can be calculated from list elements by multiplying them by \-P\-I/4. \-Direction 0 is therefore left, 1 is up-\/left, 2 is up, 3 is up-\/right and so forth. \par
 \-For example, the letter \-M would be encoded as \{2,7,1,6\} when traced left to right, and \{2,5,3,6\} when traced right to left
