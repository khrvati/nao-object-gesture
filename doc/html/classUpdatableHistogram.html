Updates an existing histogram.

Given a masked image and a learning coefficient alpha, decay the pixel accumulator by *=(1-alpha), add the masked pixels to the accumulator, then construct a new histogram from the accumulated pixels.

Parameters:
imageInput image already converted to the desired color space
alphaLearning coefficient
maskMatrix of type CV_8U which defines the pixels to consider when building the histogram.

Reimplemented from Histogram.
